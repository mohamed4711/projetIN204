# projetIN204

On devra concevoir un tracé de rayon dans une scene constituée d'objets de differentes natures et textures.
Un version minimale c'est de d'avoir une UI qui contient un bouton pour lancer l'algorithme de calcul du shadowing sur une scene minimale.
Une version avancée considérable est de paralléliser le calcul pour le rendre plus rapide: il est donc interessant d'evaluer le cout en temps du rendu.
